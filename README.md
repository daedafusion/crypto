# crypto
Crypto Helper Library

Supports RSA PKI (encryption & digital sigs) and symmetric (AES) operations as well as programmatic generation of 
key material.

## Maven

```xml
<dependency>
    <groupId>com.daedafusion</groupId>
    <artifactId>crypto</artifactId>
    <version>1.0</version>
</dependency
```
